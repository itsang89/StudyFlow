Core Features
1. Course Management
Add new courses

Edit existing courses

Delete courses

Input fields: course name, course code, instructor name, classroom/location

Assign custom color to each course (color picker)

Set recurring schedule (day of week + start/end time for each class)

View all courses in a list

2. Assignment & Deadline Tracking
Add assignments, exams, and projects

Edit existing tasks

Delete tasks

Input fields: title, linked course (dropdown), due date/time, type (assignment/exam/project), priority (low/medium/high), description/notes

Mark assignments as complete (checkbox)

View completion status

Filter assignments by: course, completion status, priority level

Smart notifications: 24 hours before assignment deadline, 1 hour before class time

3. Study Timer
Appears when user starts session from Home page

Start study session timer

Pause timer

Stop timer

Select which course you're studying (dropdown selection)

Display elapsed time (MM:SS or HH:MM:SS format)

Background timer support (continues when app minimized)

Session completion modal after stopping with: total duration display, course studied display, optional notes input field

Save completed session to history

Return to Home page after session complete

4. Calendar & Schedule Views
Monthly calendar view

Day view showing all events for selected date

Week view with grid layout

Dot indicators on calendar dates with events

Color-coded events by course

Display classes, assignments, and exams

Filter events by type: All / Classes / Assignments / Exams

Tap event to view details

Tap event to edit

Quick add event (floating action button)

Event details: title, course, date/time, location, priority, notes

5. Statistics & Analytics
Total study time (all-time)

Total study time (this week)

Study time breakdown by course (chart: bar or pie)

Study session history list with: date, course, duration, optional notes

Weekly study time trend (line chart showing last 4 weeks)

Most studied course indicator

Filter history by date range

Filter history by course

6. Settings & Personalization
User profile: name input

Theme toggle: light mode / dark mode

Manage courses (navigate to courses list)

Notification preferences with toggles:

Class reminders (on/off)

Assignment alerts (on/off)

Data management: clear all history (with confirmation dialog)

App information: version number, help/FAQ link, rate app button
